# 概要
## Tool - 1
> 更好的`OneEditor`。

### 功能
- 使`OneEditor`显示其他语言（更多文字）。

### 未来
- 将来你可以通过`python`程序选择自己国家的语言，然后自动执行，但现在你需要自己动手(这将在下一个正式版本中实现)。  
- 新增`Setting`页面，用于编辑`OneLife`的设置(`.ini`)。
- 修复`OneEditor`历史遗留问题与现存问题，[查看已知问题](https://github.com/is52hertz/LifeTools-git/issues)。
